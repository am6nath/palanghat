## 📖 About

**Palanghat Fireworks** is the official website for a well-established fireworks shop located in **Nileswaram, Kerala**, operating since 2001. The site serves as a digital storefront — showcasing the full product catalog with images, pricing, and search/filter capabilities — along with store information and direct contact options.

Built as a modern, fully responsive **React + Vite** single-page application with a refined *dark paper* aesthetic, smooth animations powered by **Framer Motion**, and a clean monochromatic design language inspired by premium print media.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎇 **Product Catalog** | Browse 70+ fireworks products across 10 categories with images and pricing |
| 🔍 **Search & Filter** | Real-time search, category filter, and price range slider |
| 🖼️ **Hero Slideshow** | Auto-advancing hero section with animated transitions |
| 📍 **Store Location** | Embedded Google Maps with store hours and contact info |

| ✨ **Scroll Animations** | Smooth entrance animations as sections come into view |
| 🎆 **Fireworks Canvas** | Interactive animated canvas backgrounds |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm v9 or higher

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/am6nath/palanghat.git
cd palanghat

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start local development server |
| `npm run build` | Build optimized production bundle |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint on the codebase |
